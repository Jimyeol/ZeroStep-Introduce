## Part 2: 컨셉 전략 (Top 3)

### 1. 한국어 특화 RAG 플랫폼

**핵심 가치(USP)**: 기업 내부 문서(PDF, HWP, DOCX)를 업로드하면 즉시 AI 검색이 가능한 노코드 플랫폼. 한국어 최적화 + 엔터프라이즈 보안 특화.

**타겟 페르소나**:
- **페르소나 A** — IT 담당자 없는 중견기업 운영팀장 (45세). 사내 규정집, 계약서를 ChatGPT에 붙여넣다가 보안 우려로 중단. 사내 배포형 솔루션 필요.
- **페르소나 B** — 스타트업 CTO (32세). LangChain으로 직접 구축하기엔 시간 부족. 빠른 MVP가 목표.
- **페르소나 C** — 법무법인 IT 책임자 (38세). 수만 건 판례/계약서 검색 자동화 필요. 데이터 외부 유출 절대 불가.

**사용 시나리오**:
1. **사내 지식베이스**: HR팀이 취업규칙 PDF 업로드 → 직원들이 "연차는 몇 일?" 질문 → 출처 조항과 함께 답변
2. **영업 지원**: 제안서 수백 개 업로드 → 영업팀이 "반도체 고객 레퍼런스 있어?" 즉시 검색
3. **법무 검토**: 계약서 업로드 → "이 계약의 손해배상 조항 요약해줘" 자동 분석

**제품 형태**: SaaS + 온프레미스 옵션
**수익 모델**: Starter $49/월 / Pro $199/월 / Enterprise 커스텀 (온프레미스 연간 라이선스)

---

### 2. VoiceForge (Fish-Speech 기반 TTS)

**핵심 가치(USP)**: 3초 음성 샘플로 즉시 클로닝. 감정·속도·억양 제어. 한국어 포함 다국어 지원.

**타겟 페르소나**:
- **페르소나 A** — 유튜브 크리에이터 (27세). 매일 나레이션 녹음이 피곤. 목소리 클로닝으로 자동화 원함.
- **페르소나 B** — e-러닝 콘텐츠 제작사 PM (35세). 강사 스케줄 맞추기 어려워 TTS 도입 검토 중.
- **페르소나 C** — 게임 개발사 (B2B). NPC 대사 수천 개를 성우 없이 제작 필요.

**사용 시나리오**:
1. 크리에이터가 스크립트 입력 → 본인 목소리 클론으로 나레이션 자동 생성
2. 기업이 고객센터 ARS 음성을 브랜드 보이스로 커스터마이징
3. 게임사가 캐릭터별 목소리 팩을 API로 대량 생성

**제품 형태**: API-first + 웹 대시보드
**수익 모델**: 문자 수 기반 종량제 (1K자당 $0.015) + 월정액 번들

---

### 3. PandaCloud (Lightpanda 기반 헤드리스 브라우저)

**핵심 가치(USP)**: AI 에이전트 전용 초경량 헤드리스 브라우저 클라우드. Puppeteer 대비 10배 빠른 실행 속도.

**타겟 페르소나**:
- **페르소나 A** — AI 에이전트 개발자 (29세). Browserbase가 너무 비쌈. 저렴한 대안 필요.
- **페르소나 B** — 데이터 파이프라인 엔지니어 (33세). 대규모 웹 크롤링 인프라 비용 절감 목표.

*(컨셉은 유효하나 라이선스 및 기술 제약으로 No-Go 판정)*

## Part 4: 최종 추천 — RAG 플랫폼 실행 로드맵

### Go/No-Go 판단 기준

| 기준 | 조건 | 상태 |
|------|------|------|
| 라이선스 | 상업 사용 가능 | ✅ Apache 2.0 |
| MVP 비용 | 1.5억 이내 | ✅ 7,500만~1억 |
| 핵심 인력 | 3인 이내 확보 | ✅ 일반 웹/ML 개발자 |
| 차별화 | 기존 대비 명확 | ✅ 한국어 특화 + 노코드 |
| 시장 검증 | 유료 고객 1곳 | ⏳ MVP 후 확인 |

### Phase 1: MVP (0~3개월) — 7,500만~1억

| 기간 | 작업 | 산출물 |
|------|------|--------|
| W1~2 | OpenRAG 포크 + 한국어 임베딩 모델 통합 | 개발 환경 |
| W3~4 | 한국어 문서 파서 최적화 (PDF, HWP, DOCX) | 파서 엔진 |
| W5~8 | 관리 UI (Next.js), 노코드 파이프라인 빌더 | 웹 대시보드 |
| W9~10 | 데이터 커넥터 4종 (Confluence, SharePoint, Notion, Slack) | 커넥터 |
| W11~12 | 베타 테스트 (파일럿 고객 2~3곳) | MVP 완성 |

### Phase 2: 시장 진입 (3~12개월)

- 유료 전환: Starter $49 / Pro $199 / Enterprise 커스텀
- 한국어 성능 최적화 (커스텀 임베딩, 리랭커)
- 엔터프라이즈 기능: RBAC, SSO, 감사 로그, 온프레미스 배포
- 목표: 고객 5~10곳 / ARR $200~400K
- SOC 2 Type II 인증 착수

### Phase 3: 스케일업 (1~3년)

- 수직 솔루션: 법률, 금융, 제조 특화 템플릿
- 커넥터 마켓플레이스 (SAP, Oracle, Salesforce)
- 해외 진출: 일본 시장 우선
- 시리즈 A 목표: ARR $1M 달성 시점
- 목표 ARR: Y2 $2M / Y3 $5~8M
